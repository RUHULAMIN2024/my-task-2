# MyTask - React Blog App

A simple blog app using React, TypeScript, Tailwind CSS, and ShadCN, fetching posts from JSONPlaceholder API.

## Features

- Fetch & display posts
- Search functionality
- Post details page
- Pagination
- Dark mode support

## Tech Stack

- React, TypeScript
- React Router, Axios
- Tailwind CSS, ShadCN
- Lucide-react (icons)

## Installation

```sh
git clone https://github.com/RUHULAMIN2024/my-task-2.git
cd my-task-2
npm install
npm run dev
```
